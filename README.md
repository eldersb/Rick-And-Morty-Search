# Rick-And-Morty-Search
 Projeto de busca de personagens de Rick and Morty, que permite fazer requisições a API, buscando informações dos personagens.

# Tecnologias Utilizadas
 * HTML, CSS
 * Bootstrap
 * JavaScript + Alpine.js
 * Axios
